![Settings Panel](../../../../assets/images/docs/settings.png)

Access via the **Gear icon** in the navigation bar. Settings are organized into sections.

| Section | What's Inside |
|---------|---------------|
| [Appearance](/docs/guide/settings/appearance/) | Language, animations, 3D preview, home widgets, display settings (theme, brightness, sleep, time format) |
| [Printer](/docs/guide/settings/printer/) | Filament sensors, AMS, Spoolman, LED settings, retraction, macro buttons |
| [Notifications](/docs/guide/settings/notifications/) | Sound settings (volume, themes, UI sounds), print completion alerts |
| [Motion](/docs/guide/settings/motion/) | Z movement style, machine limits, E-Stop confirmation, cancel escalation |
| [System](/docs/guide/settings/system/) | Network, power devices, host, printers, touch calibration, hardware health, plugins, telemetry, restart, factory reset |
| [Help & About](/docs/guide/settings/help-about/) | Debug bundles, Discord, documentation, About sub-overlay (version info, updates, print hours, branding) |
| [LED Settings](/docs/guide/settings/led-settings/) | Strip selection, auto-state lighting, macro devices, hardware setup guides |
